# 🏠 House Price Prediction App

**Live Demo:** [https://house-price-prediction-ikciucnwhdeqfz2ewgnrgd.streamlit.app](https://house-price-prediction-ikciucnwhdeqfz2ewgnrgd.streamlit.app)

An interactive AI web app that predicts house prices based on property features.

## Features
- Real-time price prediction
- Supports 7 features (Area, Bedrooms, Bathrooms, Age, Location, Parking, Floor)
- Data visualizations (Charts)
- Clean UI with Streamlit

## Tech Stack
- Python, Pandas, Scikit-learn
- Streamlit
- Deployed on Streamlit Cloud
